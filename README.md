# XIRR Financial Calculator

This project calculates the Extended Internal Rate of Return (XIRR) for a set of financial cash flows.

## What It Does
- Accepts lists of dates and cash flow amounts
- Calculates XIRR using numerical methods
- Includes edge case handling for basic inputs
